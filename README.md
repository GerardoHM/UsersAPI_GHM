# UsersAPI\_GHM

Prueba técnica para la empresa Chakray, API Restful construida con Java 17, Docker, Docker compose, springboot 3.5.14, Swagger Open API.



Instrucciones de uso:



Dentro de UsersAPI\_GHM ejecutar el comando **docker compose up --build** para proceder a construir la imagen, levantar el contenedor Docker y poder realizar pruebas con la colección de postman que se encuetra también dentro de la carpeta con el nombre de **ExamChakray.postman\_collection.**



Con Postman es primero ejecutar el login antes de todo, copiar el token y ponerlo en las autorizaciones de cada petición, pueden probar sin poner en cada petición el token para verificar que no funciona sin el.



En swagger primero se ejecuta el login se agrega lo mismo del cuerpo del login de postman, tax\_id y password, si no funciona el login de swagger se puede probar con el token de postman generado en el login, la url de swagger es la de por defecto [http://localhost:8080/swagger-ui/index.html](http://localhost:8080/swagger-ui/index.html)
